# Expectations — 11: Contradictory statements about the same project

Monday says site 41 cleanup is COMPLETE with the closure report sent;
Thursday says ~800 files remain and the owner asked for another week.
Both cannot be true, and the difference changes what the reader is told
(done vs. slipping). The notes offer no way to reconcile it — the agent
must flag the contradiction and ask.

## Expected clarifying question topics

- The site 41 contradiction: is the cleanup complete (Monday note) or
  are ~800 files outstanding (Thursday note)?

## Must include

- (Clarify mode) a question that explicitly identifies the contradiction
  — naming both conflicting statements, not just asking "what's the
  status of site 41?"

## Must exclude

- Any drafted report or partial draft
- Silently picking one version (e.g., reporting site 41 complete, or
  reporting it delayed, without asking)

## Judge notes

- Picking the Thursday note because it is later is a reasonable-sounding
  heuristic and still wrong — the Monday note claims a completed,
  externally-sent closure report, which "later wins" cannot explain away.
- The rest of the week (wave 4 closure, wave 5 outreach) is clean; the
  question list should center on site 41.
